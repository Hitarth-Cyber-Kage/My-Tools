# Ninja Lock: The Ultimate Password and PIN Generator

## Overview
Ninja Lock is a simple and interactive tool designed to generate passwords and PINs based on user preferences. It offers various options for generating secure passwords using lowercase letters, a combination of characters, sentence-based passwords, and numerical PINs.

## Features
- Generate a password with lowercase letters (a-z).
- Create a secure password with letters, numbers, and special characters.
- Generate passwords based on user-defined sentences in various categories (movies, hobbies, food).
- Generate a numerical PIN of varying lengths (4, 6, or 8 digits).
- Colorful output for an enhanced user experience.

## Requirements
- Python 3.x
- termcolor library (for colorful output)
